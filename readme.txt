Floating Social Share Bar (Lizard)  â€“ is the social sharing plugin with topmost social media link, an characteristic functionality of the floating plugin is the ability to change its background-color-bar based on the active social media selection from the plugin backend.

Contribution link: https://github.com/shshanker/floating-social-share-bar-lizard



### The Plugin Features:

* Counter - URL based total shares counter. (New)

* Lizard - The plugin has an ability to change its background color bar based on the type of social share icon displayed.

* Floating - Enables floating social share options on fixed left/right side of your site.

* Position - You can simply set the position of the share icon (right to left). Top position of the plugin has been calculated based on number of social share icon displayed. 

* Full Responsive - Tested on mobile browsers.




== Installation ==
* Download to your plugin directory or simply install via WordPress admin interface.
* Activate.
* Goto Setting admin menu, right on the button you'll find "Floating Social Share".
* Set and update as per you requirement.
* Float.
